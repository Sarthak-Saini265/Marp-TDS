# Images Directory

Place your presentation images here.

## Required Images

- `background.jpg` - Background image for slide 4 (recommended: 1920x1080)

## Supported Formats

- JPG/JPEG
- PNG
- SVG
- GIF

## Best Practices

- Optimize images before adding (use tools like TinyPNG)
- Use descriptive filenames
- Keep images under 1MB for faster loading
